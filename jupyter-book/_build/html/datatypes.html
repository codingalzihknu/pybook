5. 기초 자료형#

일반적으로 자료형으로 불리는 자료 유형data type은 사용되는 값과 관련된 많은 정보를 제공한다. 파이썬 실행기python interpreter는 자료형에 포함된 정보를 이용하여 주어진 값의 사용법을 확인하고 이용한다. 이번 장에서는 파이썬 프로그래밍 언어가 제공하는 네 종류의 기초 자료형 간단히 살펴본다.

  • 정수

  • 부동소수점

  • 부울값

  • 문자열

5.1. 정수#

일반적으로 알고 있는 -2, -1, 0, 1, 2 등의 정수(자연수, 0, 음의 정수)는 int 자료형의 값이다.

type(1234)

int

type(-872490)

int

정수는 덧셈, 뺄셈, 곱셈, 나눗셈 등의 사칙연산을 포함한 다양한 연산에 사용된다.

x = 1234 + 7329820
print(x)

7331054

x = 43214 - 7329820
print(x)

-7286606

x = 895023 * 7329820
print(x)

6560357485860

// 는 정수 나눗셈의 몫을 계산한다.

x = 7329820 // 1234
print(x)

5939

% 는 정수 나눗셈의 나머지를 계산한다.

x = 7329820 % 1234
print(x)

1094

일반적인 나눗셈은 / 가 계산한다. 하지만 실행 결과는 정수가 아닌 부동소수점이다.

x = 7329820 / 1234
print(x)

5939.886547811993

5.2. 부동소수점#

부동소수점은 1.2, 0.333, -1.2, -3.7680, 4.0 등의 유한 소수를 가리키며 float 유형의 값이다.

type(1.2)

float

type(-3.7680)

float

부동소수점은 원래 실수를 컴퓨터에서 다루기 위해 개발되었으나 컴퓨터의 한계로 실제로는 유한 소수만을 다룬다. 예를 들어 3.14, 3.141592 등으로 무리수인 원주율 \(\pi\)를 소수점 이하 적당한 자리에서 끊어서 사용한다. 소수점 이하 몇 째 자리에서 끊느냐에 따라 계산의 정밀도precision가 달라진다.

pi = 3.141592 # 원주율

정수와 마찬가지로 사칙연산 등 다양한 연산을 지원한다.

2.1 + 3.3

5.4

정수와의 연산도 가능한데, 이때 정수를 부동소수점으로 취급한다. 예를 들어 3은 3.0으로 처리된다.

2.3 + 3

5.3

2.3 + 3.0

5.3

5.3. 문자열#

문자열string은 문자 기호로 이루어진 단어, 문구 등을 가리킨다. 키보드에 포함된 영문 알파벳, 한글 자음과 모음 등을 기본적으로 사용한다. 문자열은 세 가지 방식으로 작성될 수 있다.

작은따옴표 활용

s = '안녕하세요.'
print(s)

안녕하세요.

큰따옴표 활용

s = "안녕하세요."
print(s)

안녕하세요.

연속된 세 개의 작은 또는 큰따옴표 활용

여러 줄로 이루어진 문자열 작성에 사용된다.

s = '''안녕.
잘 지내?'''

print(s)

안녕.
잘 지내?

s = """안녕.
잘 지내?"""

print(s)

안녕.
잘 지내?

스페이스와 탭을 활용해서 생성된 여백white space도 문자열로 처리된다.

s = '''안녕.
    잘 지내?'''

print(s)

안녕.
    잘 지내?

s = """안녕.
    잘 지내?"""

print(s)

안녕.
    잘 지내?

유니코드 문자

키보드 상에 존재하지 않는 다른 기호나 문자도 유니코드Unicode 형식으로 지원된다. 유니코드는 전 세계의 모든 문자를 컴퓨터에서 일관되게 표현하고 다룰 수 있도록 설계된 표준화된 체계이다. 대표적으로 서유럽 언어에서 많이 사용되는 à, â, ç, è,ü 등 특수 알파벳 등이 유니코드로 지원된다.

또한 이모티콘도 지원된다. 예를 들어 웃는 얼굴의 이모티콘을 나타내는 문자열은 다음과 같다.

>>> s = '\U0001f600'
>>> s
'😀'
>>> s * 2
'😀😀'

유니코드로 지원되는 모든 이모티콘은 Emoji Charts에서 확인할 수 있다. 단, U+로 시작하는 부분을 U000 으로 대체해서 사용해야 한다.

5.3.1. 백슬래시와 기능 해제 문자#

문자열 자체에 따옴표가 포함되면 조심해야 한다. 예를 들어, 작은따옴표가 포함된 문자열을 다음과 같이 지정하면 오류가 발생한다.

>>> s = 'Python's grammar'
  File "<stdin>", line 1
    s = 'Python's grammar'
                ^
SyntaxError: invalid syntax

이유는 작은따옴표로 문자열의 시작을 지정했기 때문에 Python's 에 사용된 작은따옴표가 문자열의 끝을 의미하게 되는데 이후에도 문자열이 이어지게 되어 결국 문자열의 끝이 불분명해져서 구문 오류를 뜻하는 SyntaxError가 발생한다.

이런 종류의 오류를 방지하는 다양한 방식이 존재하지만, 여기서는 백슬래시backslash 기호 (\)를 따옴표 바로 앞에 추가하는 방식만 소개한다.

s = 'Python\'s grammar'
print(s)

Python's grammar

슬래시 기호(\)와 원화 기호(₩)

백슬래시 키는 키보드의 Enter 키 바로 위에 위치하는데 한글 키보드의 경우 원화 기호 키 ₩ 가 대신 자리잡고 있을 수 있다. 그리고 모니터 화면에도 사용하는 운영체제의 언어 설정에 따라 원화 기호(₩)와 백슬래시 기호(\) 둘 중에 하나로만 보인다. 하지만 단순한 언어 설정의 차이일 뿐 기능은 동일하다.

백슬래시는 이처럼 특정 문자와 함께 사용되면 특별한 기능을 갖는 기능 해제 문자escape character를 구성한다. 가장 많이 사용되는 기능 해제 문자의 목록은 다음과 같다.

Example 5.1

예를 들어 백슬래시 두 개를 포함한 문자열을 지정하려면 따라서 네 개의 백슬래시를 사용한다. 즉, 백슬래시 문자를 하나의 문자로 표현하려면 항상 짝수 개의 백슬래시가 사용되어야 한다.

>>> print("\\\\")
\\

5.3.3. 화이트 스페이스#

화이트 스페이스whitespace는 화면상 보이지 않는 여백을 가리키는 문자를 의미한다. string 모듈의 whitespace 변수가 화이트 스페이스 문자를 모두 포함한다.

import string

string.whitespace

' \t\n\r\x0b\x0c'

' \t\n\r\x0b\x0c' 의 맨 왼쪽에는 스페이스 바를 한 번 눌렀을 때 발생하는 여백 문자 ' ' 이 위치한다. 위 문자열에 포함된 각 화이트 스페이스 기호의 의미는 다음과 같다.

기호

설명

' '

스페이스space

'\t'

탭tab

'\n'

줄 변경new line

\r

캐리지 리턴carriage return

'\v' 또는 '\x0b'

수직탭vertical tab

'\f' 또는 '\x0c

폼피드form feed

이중에 스페이스, 줄 변경, 탭이 가장 많이 사용되며, 다른 화이트 스페이스는 여기서는 다루지 않는다.

print(' 1\n2\t3')

 1
2	3

'' 대 ' '

빈 문자열empty string '' 는 아무런 문자도 포함하지 않았기 때문에 문자열의 길이 즉, 문자열에 포함된 문자의 개수가 0이다. 반면 ' ' 는 눈에 보이지는 않지만 스페이스 여백 하나를 포함하는 문자열이며, 따라서 길이가 1이다.

C, 자바 등의 언어에서는 길이가 1인 문자열을 character 자료형으로 별도 관리하며, 보통 char 로 표기한다. 하지만 파이썬은 character 자료형을 제공하지 않는다.

Example 5.2

화이트 스페이스가 기능을 발휘하지 않고 그 자체로 문자열에 포함되도록 하려면 앞서 언급한 백슬래시 자체를 기능 해제 문자로 표현한다.

>>> print("Hello\\n World")
Hello\n World

>>> print("Hello\\t World")
Hello\t World

5.3.4. 날 문자열#

날 문자열raw string은 ‘가공되지 않은’의 의미를 갖는 영어 단어 raw 의 첫 글자인 'r' 이 추가된 문자열이다. 기능 해제 문자가 날 문자열에 포함되더라도 특별한 기능을 발휘하지 못한다.

print(r"Hello\ World")

Hello\ World

print(r"Hello\n World")

Hello\n World

print(r"Hello\t World")

Hello\t World

5.3.5. 문자열 포매팅#

문자열 포매팅string formatting은 문자열 안에 변수 또는 표현식을 사용하여 변하는 값을 반영하여 문자열을 생성하도록 한다. 여러 방식이 있지만 여기서는 가장 최신 방법인 f-문자열f-string을 활용하는 방법을 소개한다. f-문자열은 문자열 앞에 f를 붙이며, 문자열 안에 임의의 표현식이 중괄호로 감싸여서 사용될 수 있다.

name = '강현'
age = 3
s = f'{name}이는 만으로 {age-1}살이다.'
print(s)

강현이는 만으로 2살이다.

s1 = f'{name}이의 동생은 {age - 2}살이다'
print(s1)

강현이의 동생은 1살이다

출력되는 문자열의 길이를 지정하고 좌우로 정렬할 수 있다.

s = 'hi'

print(f'{s:<10}')
print(f'{s:>10}')
print(f'{s:^10}')

hi        
        hi
    hi

남은 자리를 지정된 문자로 채울 수 있다.

print(f'{s:+<10}')
print(f'{s:0>10}')
print(f'{s:-^10}')

hi++++++++
00000000hi
----hi----

부동소수점의 소수점 이하 자릿수를 지정하려면 {표현식:.1f} 와 같은 형식을 사용한다.

num = 17.123456789

print(f'{num:.1f}')
print(f'{num:.2f}')
print(f'{num:.3f}')

17.1
17.12
17.123

전체 자릿수를 지정하려면 {표현식:3.2f} 와 같은 형식을 사용한다.

print(f'{num:15.2f}')

          17.12

좌우 정렬도 가능하다.

print(f'{num:<15.5f}')
print(f'{num:>15.5f}')
print(f'{num:0>15.5f}')

17.12346       
       17.12346
000000017.12346

5.4.1. bool 유형#

bool 유형은 참과 거짓을 의미하는 True와 False 두 개의 값만 갖는다.

type(True)

d = False
type(d)

bool

5.4.2. 논리식#

논리식은 True와 False 두 개의 값중 하나를 가리키는 표현식이다. 예를 들어, 두 표현식의 등가 여부를 판단하는 연산자 ==를 이용한 논리식은 다음과 같다.

5 == 4 + 1

True

3 + 2 ==  3 * 2

False

== vs. =

표현식 A와 B가 동일한 값을 표현하거나 가리킬 때 서로 등가equality라고 말하며, 이때 A == B를 True로 간주한다. 예를 들어 3 + 1과 2 * 2가 동일한 값을 나타낸다는 사실은 아래 코드와 같이 실행할 때 결과가 True로 나오는 것으로 확인된다.

>>> 3 + 1 == 2 * 2
True

반면에 3 + 1 == 2 * 1는 서로 등가가 아니라고 판명된다.

>>> 3 + 1 == 2 * 1
False

반면에 하나의 등호 기호 = 는 변수 할당에 사용되는 특별한 기호이다. 만약 x = 3 이 변수 x와 정수 3이 등가임을 의미한다면 3 = x 또한 의미가 있어야 한다. 그런데 이는 허용되지 않는 구문이다. 파이썬을 포함한 대부분의 프로그래밍 언어는 변수를 선언할 때 변수는 등호 기호 왼편애 위치하도록 강요한다.

>>> x = 3
>>> 3 = x
  File "<stdin>", line 1
    3 = x
    ^
SyntaxError: cannot assign to literal

두 표현식이 서로 다른 값을 가리키는지 여부, 즉 등가성의 반대를 판단하는 연산자는 !=이며, == 와 반대로 작동한다.

5 != 4 + 1

False

3 + 2 !=  3 * 2

True

5.4.3. 비교 연산자#

논리식은 기본적으로 등식, 부등식 등의 비교 연산자를 이용하여 표현하며, 논리 연산자를 이용하여 보다 복잡한 논리식을 구성한다.

비교 연산자는 수, 문자열 등의 크기를 비교할 수 있으며 아래 연산자를 지원한다.

연산 기호

의미

예시

실행 결과

<

작다

2 < 1

False

<=

작거나 같다

1 <= 2

True

>

크다

2 > 1

True

>=

크거나 같다

1 >= 2

False

==

같다

3 == 2 * 1

False

!=

같지 않다

1 != '1'

True

두 값의 등가 여부를 판단하는 == 연산자는 자료형이 다르면 무조건 거짓으로 판정한다.

1.0 == '1.0'

False

하지만 정수는 부동소수점으로 취급하기에 예를 들어 1과 1.0은 등가라고 판정한다.

1 == 1.0

True

문자열도 비교 연산자를 지원하며 사전식 순서 개념을 사용한다.

'apple' < 'banana'

True

영어 알파벳의 경우 대문자가 소문자보다 작다고 판단한다.

'apple' < 'Apple'

False

Example 5.3

변수 x가 가리키는 값이 짝수이면 2로 나눈 값을, 그렇지 않으면 두 배한 값을 출력하도록 해보자.

정수의 짝수 여부는 2로 나눈 나머지가 0 또는 1인지 여부로 판단할 수 있다. 파이썬의 나머지 연산자는 % 이다.

>>> 21 % 6
3
>>> 21 % 4
1

따라서 다음 논리식이 짝수 여부를 판단한다.

x % 2 == 0

반면에 몫을 계산하는 연산자는 // 이다.

>>> 27 // 6
4
>>> 27 // 4
6

정리하면 짝수일 때와 홀수일 때 지정된 일을 하도록 하는 명령문은 다음과 같다.

if x % 2 == 0:
    print(x // 2)
else:
    print(x * 2)

5.4.4. 논리 연산자#

영어의 and, or, not 의 개념과 거의(!) 유사하게 작동하는 세 개의 논리 연산자를 이용하여 보다 복잡한 논리식을 구현할 수 있다.

연산 기호

의미

예시

실행 결과

and

그리고

1==2 and 3==2+1

False

or

또는

1==2 and 3==2+1

True

not

부정

not 1==2

True

논리 연산자들 사이의 우선순위는 not 이 가장 높고 and 와 or 는 동등하다. 예를 들어

not a > b and b > c

는

(not a > b) and b > c

와는 서로 등가이지만

not (a > b and b > c)

와는 일반적으로 서로 다른 값을 가리킨다.

파이썬이 제공하는 편리 기능 중에 하나로 논리 연산자의 인자로 논리식이 아닌 값이 사용될 수 있다. 예를 들어, 0이 아닌 수는 참, 0은 거짓으로 처리된다.

17 and True

True

이외에 None은 거짓, 빈 리스트는 거짓으로 그렇지 않은 리스트는 참으로 등등처럼 많은 값들에 대해 비어 있거나 또는 의미가 없는 값은 거짓으로 그렇지 않으면 참으로 처리한다. 하지만 이런 방식은 코드의 이해를 어렵게 만들 수 있기에 조심해서 사용해야 한다.

Example 5.4

세 개의 수 중에서 최댓값을 출력하는 코드는 다음과 같다.

a = 3
b = 7
c = -1

if a > b and a > c:
    print(a)
elif a > b and not a > c:
    print(c)
elif not a > b and b > c:
    print(b)
else:
    print(c)

Example 5.5

세 개의 수가 주어졌을 때 양수가 최소 하나라도 있는 경우에만 True를 출력하도록 하는 코드는 다음과 같다.

a = -23
b = -7
c = 1

if a > 0 or b > 0 or not c <= 0:
    print(True)
else:
    print(False)

참고

and 와 or 가 자연어의 and 와 or 와는 조금 다르게 작동한다. 예를 들어, 아래와 같이 False and 3/0를 실행하면 False로 계산되는데 좀 이상하다. 왜냐하면 3/0 이 ZeroDivisionError 라는 오류를 발생시켜야 할 것으로 기대되기 때문이다.

>>> False and 3/0
False

실제로 3/0 and False 를 실행하면 3/0을 먼저 확인하기에 오류가 발생한다.

>>> 3/0 and False
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ZeroDivisionError: division by zero

False and 3/0 를 실행할 때 오류가 발생하지 않은 이유는 x and y는 x가 참일 때만 y 를 확인하기 때문이다. x 를 먼저 확인해서 거짓으로 판명되면 논리식 전체를 바로 거짓으로 계산한다. 반면에 x or y는 x가 거짓일 때만 y를 확인한다. 즉, x를 먼저 확인해서 참이되면 논리식 전체를 참으로 계산한다.

5.5. 형변환#

2.3 + 3을 실행하면 파이썬 실행기interpreter는 내부적으로 정수를 부동소수점으로 형변환하는 함수 float()를 이용하여 정수 3을 부동소수점 3.0으로 유형을 변환한 후에 덧셈을 처리한다. 이렇게 값의 유형을 적절하게 자동 변환하는 기능을 형변환type casting이라 부른다.

float(3)

3.0

(2.3 + 3) == (2.3 + float(3))

True

float() 함수 이외에 int(), str(), bool() 등 다양한 형변환 함수가 제공된다.

float 대 float()

float는 부동소수점 자료형을 가리키기도 하고 형변환 함수를 가리키기도 한다. 여기서는 자료형과 함수 이름을 구분하기 위해 함수는 int(), float(), str() 등처럼 함수 이름과 괄호를 함께 사용하는 표기법 관행을 따른다.

int() 함수

정수 모양의 문자열 또는 부동소수점을 정수로 변환한다.

int('5')

5

int(4.8)

4

True와 False는 각각 1과 0으로 취급된다.

int(True)

1

int(False)

0

float() 함수

유한소수 모양의 문자열 또는 정수 등을 부동소수점으로 변환한다.

float(7)

7.0

float('7')

7.0

float('7.9')

7.9

주의

int()함수의 인자로 문자열을 사용할 때는 그 모양이 정수의 모양을 갖추어야 한다.

>>> int('5.1')
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: invalid literal for int() with base 10: '5.1'

float() 함수의 문자열 인자는 정수 또는 부동소수점 모양이어야 한다.

>>> float('5.2a')
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: could not convert string to float: '5.2a'

str() 함수

임의의 값을 문자열로 변환한다.

str(6)

'6'

str(7.6)

'7.6'

str(False)

'False'

bool() 함수

임의의 값을 True 또는 False 로 변환한다. 0, 0.0, ''(빈 문자열) 등처럼 0, 비어 있는 것, 의미 없는 것등은 False 로 그렇지 않으면 True 로 지정한다.

bool(0)

False

bool(2)

True

bool(0.0)

False

bool(0.01)

True

bool('')

False

bool('Hello')

True

5.6. 기타 자료형#

파이썬은 많은 자료형을 제공한다. 정수, 부동소수점, 문자열, 부울값 등 기초 자료형 이외에 파이썬이 함께 기본으로 제공하는 자료형은 %s장에서 자세히 다룰 리스트, 튜플, 사전, 집합 등의 모음collection 자료형이 있다.

파이썬에 추가 모듈 또는 패키지를 설치하면 더 많은 자료형이 함께 제공된다. 그리고 필요에 따라 직접 새로운 자료형을 정의해서 사용할 수 있는 기능도 파이썬이 제공한다. 자료형을 직접 정의하는 방법은 나중에 21장에서 클래스를 다룰 때 자세히 살펴볼 것이다.
